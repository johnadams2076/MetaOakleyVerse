# SDD: iOS Meta Oakley / Meta AI Glasses Vision Assistant

Date: 2026-05-22
Project name: MetaOakleyVision
Platform: iOS only

This repository implements an iOS-first MVP with:
- FastAPI backend with mock photo description endpoint
- SwiftUI app generated via XcodeGen
- Mock glasses photo capture flow
- Mock AI and backend client modes
- iOS macOS-runner CI build and visual UI test artifacts

Key rule:
- Codespaces is for editing.
- iOS build/test runs on GitHub Actions macOS runners.
- Scope is iOS only.
